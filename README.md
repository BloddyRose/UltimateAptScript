# Ultimate Apt Script 

### Installation 
    ** chmod +x run.sh
    ** ./run.sh
    ** Run run.sh (recommended) if is an error then try commands  
    ** sudo pip3 install -r requirements.txt 
    ** python3 ultimateapt.py (SUDO ask when needed)


### Features 
    ** Save output to files 
    ** Easy work with apt 
    ** No more sudo apt-get install
